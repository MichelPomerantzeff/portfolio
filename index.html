Projects

Movie App

Movie App utilising React Hooks, Context API and Local Storage.

  • This App allows the user to search for movies by name, check extended information about a movie, add movies to watchlist or watched movies and delete movies from a list.
  • I used React Hooks to consume data from OMDb API as well as handle its states, Local Storage to store data of the movies added or deleted from the lists and Context API to access global states on multiple components throughout the whole application.
  • Future implementation: Add a user's own rate and comment about movies that have been watched.

    • Countdown

    It decreases time from a pre-set time to zero.

  • The set-time button displays arrows on the browser enabling the user to set the time and it only functions if the current time is equal to zero or time is paused.
  • The start button hides the arrows, updates the time and switches to a pause button.
  • The pause button stops the time, enables the set-time button to function again and switches back to start button.
